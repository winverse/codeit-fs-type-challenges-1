# 5-06. Intersection 타입

## 목표

공통 속성 조각과 역할 조합을 다시 합치고, 같은 조합 방식을 전투 캐릭터에도 적용합니다.

## 해야 할 일

1. `template.ts`를 열고 기본 문제부터 순서대로 풉니다.
2. 본문에서 정리한 기준에 맞게 타입을 바로잡고, 추가 문제까지 같은 방식으로 완성합니다.
3. `test-cases.ts`로 타입 검사를 통과시킵니다.
4. 풀이를 마친 뒤에만 해설 위치의 파일과 비교합니다.

## 파일 구성

- `template.ts`: 학생용 문제 파일
- `test-cases.ts`: 타입 검사 기준
- 해설 위치: `type-challenge/solution/5-06.Intersection-타입/solution.ts`

## 실행 방법

```bash
pnpm --package=typescript dlx tsc --noEmit type-challenge/5-06.Intersection-타입/test-cases.ts
```
