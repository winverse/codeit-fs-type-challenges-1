# 4-01. Interface

## 목표

공통 구조와 확장 구조를 다시 분리하고, 같은 분리 방식을 계정 도메인에 적용합니다.

## 해야 할 일

1. `template.ts`를 열고 기본 문제부터 순서대로 풉니다.
2. 본문에서 정리한 기준에 맞게 타입을 바로잡고, 추가 문제까지 같은 방식으로 완성합니다.
3. `test-cases.ts`로 타입 검사를 통과시킵니다.
4. 풀이를 마친 뒤에만 해설 위치의 파일과 비교합니다.

## 파일 구성

- `template.ts`: 학생용 문제 파일
- `test-cases.ts`: 타입 검사 기준
- 해설 위치: `type-challenge/solution/4-01.Interface/solution.ts`

## 실행 방법

```bash
pnpm --package=typescript dlx tsc --noEmit type-challenge/4-01.Interface/test-cases.ts
```
