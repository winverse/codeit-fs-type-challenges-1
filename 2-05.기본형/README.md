# 2-05. 기본형

## 목표

기본형 다섯 가지를 값의 특징으로 구분하고, 특히 `null`, `undefined`, `NaN` 같은 헷갈리는 값을 정확히 분류합니다.

## 해야 할 일

1. `template.ts`를 열고 기본 문제부터 순서대로 풉니다.
2. 각 값이 어떤 기본형에 속하는지 판단해 `unknown`을 알맞은 타입으로 바꿉니다.
3. `test-cases.ts`로 타입 검사를 통과시킵니다.
4. 풀이를 마친 뒤에만 해설 위치의 파일과 비교합니다.

## 파일 구성

- `template.ts`: 학생용 문제 파일
- `test-cases.ts`: 타입 검사 기준
- 해설 위치: `type-challenge/solution/2-05.기본형/solution.ts`

## 실행 방법

```bash
pnpm --package=typescript dlx tsc --noEmit type-challenge/2-05.기본형/test-cases.ts
```
