# 5-01. 리터럴 타입

## 목표

리터럴 추론과 widening을 다시 비교하고, 문자열 외의 리터럴에서도 같은 차이를 확인합니다.

## 해야 할 일

1. `template.ts`를 열고 기본 문제부터 순서대로 풉니다.
2. 본문에서 정리한 기준에 맞게 타입을 바로잡고, 추가 문제까지 같은 방식으로 완성합니다.
3. `test-cases.ts`로 타입 검사를 통과시킵니다.
4. 풀이를 마친 뒤에만 해설 위치의 파일과 비교합니다.

## 파일 구성

- `template.ts`: 학생용 문제 파일
- `test-cases.ts`: 타입 검사 기준
- 해설 위치: `type-challenge/solution/5-01.리터럴-타입/solution.ts`

## 실행 방법

```bash
pnpm --package=typescript dlx tsc --noEmit type-challenge/5-01.리터럴-타입/test-cases.ts
```
