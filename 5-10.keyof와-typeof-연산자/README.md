# 5-10. keyof와 typeof 연산자

## 목표

값에서 타입을 가져오고 다시 키를 뽑는 흐름을 반복하고, 필터 설정 객체로 같은 관계를 다시 확인합니다.

## 해야 할 일

1. `template.ts`를 열고 기본 문제부터 순서대로 풉니다.
2. 본문에서 정리한 기준에 맞게 타입을 바로잡고, 추가 문제까지 같은 방식으로 완성합니다.
3. `test-cases.ts`로 타입 검사를 통과시킵니다.
4. 풀이를 마친 뒤에만 해설 위치의 파일과 비교합니다.

## 파일 구성

- `template.ts`: 학생용 문제 파일
- `test-cases.ts`: 타입 검사 기준
- 해설 위치: `type-challenge/solution/5-10.keyof와-typeof-연산자/solution.ts`

## 실행 방법

```bash
pnpm --package=typescript dlx tsc --noEmit type-challenge/5-10.keyof와-typeof-연산자/test-cases.ts
```
